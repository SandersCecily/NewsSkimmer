# NewsSkimmer

## Intro
This app is a scraper to visit a Google page about Google related news then scrapes that page for the top search results and presents them for the user to read.

## Technology Used
* Axios
* Express
* Cheerio
* Mongoose
* Path
* mLab
* Heroku
* Handlebars
